# BdxCalendar

## Requirements
  - PHP 5 >= 5.3.0

## Installation
  - Copy all files to a folder where PHP can be executed

  - [Bootstrap](getbootstrap.com) --> [github](https://github.com/twbs/bootstrap)

  - [clipboard.js](https://clipboardjs.com/) --> [github](https://github.com/zenorocha/clipboard.js/)
